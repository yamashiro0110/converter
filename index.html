converter

ログファイルをCSVに変換する

Python2.7推奨

Install

git clone https://github.com/yamashiro-r/converter.git

Usage

cd $INSTALL_DIR
python _run_convert.py $LOG_FILE_PATH $LOG_FORMAT $OUT_PUT_CSV_PATH
  • LOG_FILE_PATH
    • 読み込むログファイルのパス
  • LOG_FORMAT
    • 読み込むログファイルのフォーマット
  • OUT_PUT_CSV_PATH
    • CSVに変換したログの出力先
